# FIFA
A quick and dirty analysis on FIFA Game data

Just Run **index.Rmd** inside Rstudio

##WARNING##

Check if the libraries mentioned below are installed:

- rio
- broom
- knitr
- prettydoc
- plotly
- RColorBrewer
- kable
- kableExtra
- summarytools
- jsonlite
- maps
- htmltools
- hrbrthemes
- DataExplorer
- tidyverse
- viridis
